## 项目构建计划

### 第一阶段：项目初始化
1. 使用 Vite 创建 React + TypeScript 项目
2. 安装依赖包：`tailwindcss`, `react-router-dom`, `lucide-react`, `classnames`, `react-markdown`
3. 配置 Tailwind CSS 和 TypeScript 严格模式

### 第二阶段：目录结构创建
创建以下目录结构：
```
src/
├── app/           # 应用初始化层
├── components/    # 通用组件
├── features/      # 业务模块
│   ├── news/
│   ├── tools/
│   ├── prompts/
│   ├── projects/
│   └── search/
├── layouts/       # 布局组件
├── shared/        # 公共资源
│   ├── types/
│   ├── hooks/
│   ├── utils/
│   ├── constants/
│   └── store/
└── data/          # 本地模拟数据
```

### 第三阶段：核心基础设施
1. **shared/types/** - 定义所有数据类型（BaseEntity, NewsItem, ToolItem, PromptItem, ProjectItem）
2. **shared/utils/search.ts** - 泛型搜索工具
3. **shared/hooks/useDocumentTitle.ts** - SEO 预留 hook
4. **shared/constants/** - 公共常量

### 第四阶段：数据层
创建模拟数据文件：
- `data/news.ts`
- `data/tools.ts`
- `data/prompts.ts`
- `data/projects.ts`

### 第五阶段：布局与通用组件
1. **layouts/MainLayout.tsx** - 主布局（含 Navbar）
2. **components/** - 通用组件（Card, MarkdownRenderer 等）

### 第六阶段：业务模块
每个模块包含：
- types.ts
- components/
- pages/
- api.ts（数据适配层）
- index.ts

模块：news, tools, prompts, projects, search

### 第七阶段：路由与应用入口
1. **app/router.tsx** - 路由配置（懒加载）
2. **app/providers.tsx** - Provider 包装
3. **App.tsx** 和 **main.tsx** - 入口文件

### 第八阶段：验证
- 运行项目确保无 TypeScript 错误
- 验证所有页面可访问
